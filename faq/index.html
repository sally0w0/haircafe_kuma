FAQよくある質問

予約は必要ですか？
予約優先となっておりますので、お電話かネットでの予約をしていただくと確実にご案内ができます。
ご予約はこちらからネット予約もできます。
当日の予約でも可能ですか？
予約の空き状況次第で当日でも可能です。
シャンプーなど商品だけ買いに行くことは可能ですか？
もちろん、お買い求め頂けます。
当店ではどのメーカーの商品でも買うことが可能です。お店に在庫がないものや取り扱いのない商品に関しましても、注文販売なども承っております。
ディーラーでの扱いがない場合もございますのでご了承ください。
前髪だけカットしてもらえますか？
もちろん、前髪カットだけでも大丈夫です。メンテナンスカットになりますので6ヶ月以内に当店でカットをされた方を対象としております。
前髪だけストレートパーマをかけれますか？
かけられます。料金は￥4,320～です。
根元の伸びた部分だけカラーできますか？
できます。全体の根元カラーメニューを選んでいただければと思います。
ただカラーも根元のみは基本メンテナンスとして考えておりますので当店初めての方やカラー久しぶりの方には全体カラーでさせて頂いております。 2〜3週間で根元の白髪が気になる方は、Tゾーンカラー(メンテナンス）〈顔回りの生え際、分け目部分〉メニューもございます。
髪型やカラーの直しなどは可能ですか？
当店ではお客様の満足度を大切にしていきたいと思っています。
仕上がりがすこしでも気になるようでしたらお気軽にご連絡ください。お直しの期限も設けておりませんので気になったらご連絡ください。
その時に、しっかりとカウンセリングさせて頂き判断させて頂いております。
クレジットカード、電子マネーは使えますか？
はい。どちらもご利用可能です。現在ご利用頂けるクレジットカード、電子マネーは
VISA / Master / JCB / Diners / AMERICANEXPRESS / DISCOVER / 交通系電子マネー/ nanaco / waon / paypay / auPAY / d払い
です。
子供を連れて行ってもいいですか？
もちろんです。当店にはキッズスペースがございます。
多くのお子様たちに「また遊びに行きたい。」と言ってもらえております。
席と近い場所にキッズスペースがございますので安心して施術が受けられます。まだキッズスペースで遊べない年齢のお子様は席の横にベビーカーを置いていただく方もいます。
ヘアセット、ヘアアレンジはやっていますか？
はい。ぜひおまかせください。
指名料や特別な料金はございますか？
指名料はございません。
※カットのみランク料金がございます。トップスタイリスト、ディレクターと料金が違うのでカットメニューはその料金でさせて頂いております。
